# Hardware and system requirements

1. Windows Client minimum requirements:
   * Windows 10 64-bit Operating system
   * CPU Minimum 2 Cores
   * Minimum 4GB of RAM
   * .NET framework 4.5 or higher
   * Minimum 15GB of free disk space
2. MacOS Client minimum requirements:
   * MacOS 10.x
   * Minimum 4GB of RAM
   * Minimum 15GB of free disk space
